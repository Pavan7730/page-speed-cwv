Privacy Policy

Page Speed & Core Web Vitals

This Privacy Policy explains how the Page Speed & Core Web Vitals Chrome extension handles user data.

Data Collection

This extension does not collect, store, transmit, or sell any personal or identifiable user data.

How the Extension Works

The extension measures Core Web Vitals metrics such as:

  • Largest Contentful Paint (LCP)
  • Cumulative Layout Shift (CLS)
  • Interaction to Next Paint (INP)

These metrics are calculated locally in the user's browser using standard browser performance APIs and are displayed only to the user.

Permissions

The extension uses the activeTab permission only when the user clicks the extension icon. This allows the extension to measure performance metrics for the currently open webpage.

Remote Code

The extension may load the open-source web-vitals library from a trusted public CDN solely to calculate performance metrics. No data is sent to any external server.

Third-Party Sharing

This extension does not share any user data with third parties.

Changes to This Policy

This privacy policy may be updated if the extension’s functionality changes. Any updates will be reflected on this page.

Contact

If you have questions about this privacy policy, you may contact the developer via the Chrome Web Store listing.
